公考计时器 - 使用说明
====================

【窗口操作】
- 拖拽窗口任意位置移动窗口
- 拖拽窗口边缘/角落缩放窗口（字体自动适配）
- 右键窗口弹出功能菜单

【计时操作】
- 窗口显示：模式 | 当前模块 | 剩余时间 | 进度条
- 右键菜单或快捷键控制计时

【快捷键】（可在设置中自定义）
  空格键               → 结束当前模块 + 切换下一模块（固定，最常用）
  Ctrl+Shift+Space   → 开始 / 暂停
  Ctrl+Shift+R       → 重置
  Ctrl+Shift+M       → 切换行测 / 申论模式
  Ctrl+Shift+T       → 切换穿透模式（鼠标点击穿透到下层窗口）
  Ctrl+Shift+F       → 切换专注模式（只显示时间数字）

【考试模式】
  行测模式 - 120 分钟整卷，5 个模块：
    常识判断 (10分钟) → 言语理解 (35分钟) → 数量关系 (15分钟)
    → 判断推理 (35分钟) → 资料分析 (25分钟)
    各模块时长可在设置中自定义

  申论模式 - 150/180 分钟可选，小题作答 + 大作文分段计时

  空格键一键完成当前模块并切换至下一模块，模块间无缝衔接

【提醒功能】
- 倒计时剩余 5 分钟、3 分钟、1 分钟时窗口闪烁提醒
- 计时结束时弹窗提醒
- 提醒可在设置中独立开关

【专注模式】
- 隐藏模式标签、模块名、设置按钮
- 只显示时间数字和进度条
- 适合做题时最小干扰

【穿透模式】
- 鼠标点击穿透到下层窗口
- 此时只能通过快捷键操作
- 适合计时器悬浮在刷题软件上方

【单题计时器】
- 右键菜单打开独立小计时器
- 用于单题计时，不影响整卷计时

【数据复盘】
- 自动记录每次刷题的整卷用时和各模块用时
- 统计超时次数、各模块平均用时
- 支持导出 Excel (.xlsx) 和 CSV (.csv)
  （右键菜单 → 导出数据，或在历史记录面板中导出）

【数据存储】
- 所有数据保存在 %APPDATA%\ExamTimer\ 目录
- config.json：用户设置（快捷键、外观、提醒、模块配置）
- history.json：历史刷题记录
- 纯本地存储，不上传任何数据

【防锁屏】
- 计时运行期间自动阻止屏幕锁屏和休眠
- 计时停止后恢复正常

【深色/浅色主题】
- 右键菜单切换深色/浅色护眼模式
- 深色模式适合长时间刷题
